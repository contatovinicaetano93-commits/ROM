# ROM Academy

Land page da Masterclass Mechas Criativas (`https://rom-academy.vercel.app`).

## Tesoura flutuante

O componente `FloatingScissors` segue o mouse no desktop e abre/fecha as lâminas conforme o scroll.

## Dev

```bash
cd academy
npm install
npm run dev
```

Abre em `http://localhost:3010`.

## Deploy (Vercel)

No projeto Vercel **rom-academy**, use:

- **Root Directory:** `academy`
- **Framework:** Next.js
- Repo: `ROM_BRASIL` (este monorepo)

Ou, a partir desta pasta:

```bash
npx vercel --prod
```
